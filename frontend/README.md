# Angular 19 Standalone App

This is a simple standalone Angular 19 application. The project demonstrates the use of Angular's standalone components approach introduced in version 14 and further enhanced in version 19.

## Features

- Built with Angular 19
- Uses standalone components
- Implements Angular routing
- Modern, responsive UI

## Development Setup

### Prerequisites

- Node.js and npm

### Installation

```bash
# Install dependencies
npm install
```

### Development Server

```bash
# Start the development server
npm start
```

Navigate to `http://localhost:4200/` to view the application.

### Build

```bash
# Build the project for production
npm run build
```

The build artifacts will be stored in the `dist/` directory.

## Project Structure

- `src/app/components/` - Reusable UI components
- `src/app/services/` - Services for data management and business logic
- `src/app/home/` - Home page component
- `src/assets/` - Static assets like images and fonts
